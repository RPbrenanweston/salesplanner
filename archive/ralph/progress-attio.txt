## Codebase Patterns
- OAuth pattern: Copy GmailOAuthButton.tsx / SalesforceOAuthButton.tsx — these are the canonical templates
- CSRF nonce: frontend/src/lib/oauth-csrf.ts — use generateNonce() and verifyNonce() exactly as Salesforce does
- Toast pattern: import { toast } from '../hooks/use-toast' with variant: 'destructive' for errors
- Supabase query pattern: always .eq('org_id', orgId) for org-scoping
- ON CONFLICT: use .upsert(..., { onConflict: 'column_name', ignoreDuplicates: true }) in Supabase JS
- Token storage: oauth_connections table with provider, access_token, refresh_token, expires_at, user_id, org_id
- TypeScript: Record<string, unknown> for unknown shapes, String(prop) for JSX from unknown
